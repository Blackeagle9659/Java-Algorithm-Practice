### How it Works:
1.  **Iteration:** The program iterates through numbers from 2 to 100 using a primary loop.
2.  **Assumption:** Every number is initially assumed to be prime (`isPrime = true`).
3.  **Verification:** A nested loop checks if the current number is divisible by any integer smaller than itself.
4.  **Optimization:** If a divisor is found, the inner loop terminates immediately using the `break` keyword to save computational resources (CPU cycles).
